# 📜 Ledger of Coins

### Forge Expeditions. Track Treasures. Settle Debts.

Ledger of Coins is a fantasy-inspired expense management platform that transforms traditional bill splitting into an immersive adventure.

Instead of managing expenses through spreadsheets, chats, and scattered notes, users can forge expeditions, record expenses, track debt scrolls, collect relics, unlock achievements, and preserve their financial history within a beautifully crafted ancient ledger.

Built with modern web technologies and an immersive fantasy aesthetic, Ledger of Coins combines practical financial management with engaging storytelling elements.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ledger-of-coins.git
cd ledger-of-coins
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=YOUR_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```

### Run the Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
