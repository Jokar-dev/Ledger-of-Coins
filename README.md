# 📜 Ledger of Coins

### Forge Expeditions. Track Treasures. Settle Debts.

Ledger of Coins is a fantasy-inspired expense management platform that transforms traditional bill splitting into an immersive adventure.

Users can forge expeditions, record expenses, track debt scrolls, collect relics, unlock achievements, generate PDF ledgers, and preserve their financial history within an ancient magical ledger.

---

## ✨ Features

### 🏛 Expedition Management

* Create and manage expeditions
* Add destinations and party members
* Track shared costs
* Organize group adventures

### 📜 Treasury Ledger

* Record personal and shared expenses
* Categorize spending
* View complete financial history
* Track expedition costs

### ⚖ Debt Scroll System

* Automatically calculate balances
* Track who owes whom
* Settle outstanding debts
* Maintain transparent expense sharing

### 🏺 Relic Collection

* Store expedition discoveries
* Add custom descriptions and values
* Associate relics with expeditions

### 🏆 Achievement System

Unlock achievements through real actions:

* Chronicle Awakened
* Expedition Founder
* First Coin Spent
* Relic Hunter
* Merchant King

### 🔮 Oracle Chamber

* Spending analytics
* Financial insights
* Expense trends
* Expedition statistics

### 📚 PDF Export

Generate downloadable reports containing:

* Expenses
* Expeditions
* Debt Scrolls
* Relics

### 👤 Explorer Profiles

* Profile picture
* Explorer title
* Chronicle name
* Biography
* Personal statistics

---

## 🛠 Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Supabase
* PostgreSQL
* Row Level Security (RLS)

## 🚀 Installation

```bash
git clone https://github.com/Jokar-dev/Ledger-of-Coins.git

cd Ledger-of-Coins

npm install

npm run dev
```

Open:

```text
http://localhost:3000
```
